Visualization Design

  • A central world map that is interactive w/ a bar chart of happiness indices
  • Surrounded by sort of "dashboard" with auxiliary data
  • Make some sketches and optimize until the final version

    • Must-Have Features

  • In order to present happy index for any individual country, an interactive map of the world is required.
  • Also, we need a way to specify or select any individual country in the map.
  • To measure/show the changes of happiness, we need to respond to selection of dates for the visualization
  • We also need brushing/other means of selection if possible

    • Optional Features

  • Map should have interactive zoom features to highlight the country
  • Ability to build visualizations based on country selections
  • Transitions for adding, removing charts
